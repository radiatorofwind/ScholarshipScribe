ScholarshipScribe

"Free money!!!"

Welcome to ScholarshipScribe, the super easy way to quickly acquire as many scholarships as possible.

Mad at the system? This website is for you! And the best part is, this can be done by asking you 5 easy to answer questions.


So, what're you waiting for? Click the button to get started!

STEP 1

"Just getting started"

For step 1, we will need your e-mail address.

This is so that we can send your resume; we will not store this forever, obviously.
Type it in the box below.

Once you've entered it, click "Validate" to make sure the e-mail address is correct.
When it is, the "Next!" button will enable.

STEP 2

"We're getting somewhere"

For step 2, we need your resume so that we can actually send it.

Simply upload it with the "Choose File" button below.
You must upload a .PDF file, otherwise it will reject it.



STEP 3

"Halfway there-ish!"

Now that we have the information we need, it's time to select what you want!

In this step, you need to select what type of scholarship you want.




STEP 4

"Nearly there!"

Well done so far. This is the last page before you can submit. Now is the time to finetune!

Use the selection boxes to finetune your messaging. When it's all done, hit "I'm Done!"







STEP 5

"You're all done!"

Congratulations! You have finished submitting information. This last step is very simple.

Simply press the "Send!" button. The console window will tell you where it's being sent to.
